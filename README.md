# CoDev-Bench

## Overview
CoDev-Bench, the first benchmark designed to evaluate repository-level code generation under collaborative development settings. Constructed from authentic PR lifecycles, CoDev-Bench comprises 244 tasks that incorporate multi-turn reviewer feedback and historical error traces, challenging models to refine solutions based on collaborative signals.

## Setup
```bash
# Clone the repository
git clone https://github.com/your-username/CoDev-Bench.git
cd CoDev-Bench

# Run the default benchmark evaluation pipeline
bash run.sh
```
